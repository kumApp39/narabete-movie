# ならべてムービー

特別支援学校の児童向け 動画作成Webアプリ(プロトタイプ)。
撮影したクリップが「カード」になり、指で並べ替えて上から順に連続再生できます。

## 使い方

- iPad の Safari で公開URLを開く: https://kumapp39.github.io/narabete-movie/
- 「ホーム画面に追加」するとフルスクリーンで使えます
- カメラが起動しない場合: 設定 > Safari > カメラ の許可を確認

## 大事なこと

- 撮影した動画は **端末の中だけ** に保存され、外部には一切送信されません
- このプロトタイプ版は、ページを閉じるとデータが消えます(本番版で端末内保存を実装予定)
