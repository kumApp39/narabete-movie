# ならべてムービー

特別支援学校の児童向け 動画作成Webアプリ。
撮影したクリップが「カード」になり、指で並べ替えて上から順に連続再生できます。
文字カード(スライド)を挟んだり、動画の前後をカット(トリミング)することもできます。

## 使い方

- iPad の Safari で公開URLを開く: https://kumapp39.github.io/narabete-movie/
- 「ホーム画面に追加」するとフルスクリーンで使えます
- カメラが起動しない場合: 設定 > Safari > カメラ の許可を確認

## みんなで送り合う (AirDrop)

- 「おくる」で送りたいカードを選び、共有メニュー → AirDrop で相手の iPad へ
- 受け取る側は AirDrop を受け入れ →「もらう」で届いたファイルを選ぶとカードに並びます
- **Wi-Fi は不要**。端末どうしを直接つなぐ AirDrop で送受信し、劣化しません

## 大事なこと

- 撮影した動画は **端末の中だけ** (IndexedDB) に自動保存され、ページを閉じても残ります
- 外部サーバーには一切送信されません(送受信は端末間の AirDrop のみ)
- 先生メニュー(⚙️長押し→1111)で「1本にする」と、ならべた順で1本の動画にして端末へ保存できます
